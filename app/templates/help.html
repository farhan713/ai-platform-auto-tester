Quick start

  1. Go to New Run
  2. Enter the Celerant tenant Login URL, username, and password
  3. Upload your questions Excel (column A = NL question, column B = optional expected SQL)
  4. Click Start QA Run
  5. Watch the live progress — click any query to drill down
  6. When done, click Open final report

Excel format

Both test types use the same xlsx shape — column A holds the test input, column B is optional metadata.

Status meaning

  • PASS Everything green
  • PARTIAL At least one warning (e.g. zero rows, mismatched columns)
  • FAIL A hard failure (call missing or HTTP error)
  • TIMEOUT Call did not return within the per-query budget

Troubleshooting

  • "Machine ID is empty" — wrong Machine ID. Try 100 or get the right value from the client.
  • Lots of TIMEOUTs — Slow LLM endpoint. Bump run-sql timeout to 300000–600000 (5–10 min).
  • Login fails immediately — Check the login URL is reachable from this machine. Try opening it in a browser first.
